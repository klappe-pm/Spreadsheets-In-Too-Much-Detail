# QUERY

## QUERY Description

Runs a Google Visualization API Query Language query across data. One of the most powerful functions in Google Sheets for data manipulation.

> [!f(x)] QUERY Syntax
>
> ```spreadsheets
> QUERY()
> ```

> [!f(x)] QUERY Example
>
> ```spreadsheets
> QUERY() → result
> ```

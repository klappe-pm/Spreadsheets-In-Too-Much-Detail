# RAND

## Description

**RAND** returns a random decimal number between 0 (inclusive) and 1 (exclusive). Each time the worksheet recalculates, RAND generates a new random value. RAND() might return 0.458291, 0.000001, or 0.999999—any value in the range [0, 1) with equal probability. This is a uniform random distribution.

This function is the foundation of **randomization in spreadsheets**. Need to randomly sort data? Assign RAND() to each row and sort by that column. Need to simulate coin flips? Use IF(RAND()<0.5, "Heads", "Tails"). Need random numbers in a different range? Transform with MIN + RAND()*(MAX-MIN). RAND is the building block from which other random behaviors are constructed.

**Critical behavior: RAND recalculates constantly.** Every time you edit any cell, press F9, or the sheet recalculates, all RAND values change. This volatility is by design but can be surprising. If you need to preserve random values, immediately copy and paste-values the cells. Otherwise, your "random" data keeps changing.

The random numbers are **pseudo-random**, generated by an algorithm seeded by system state. They pass statistical tests for randomness and are suitable for simulations, sampling, and games—but should not be used for cryptographic purposes. For most spreadsheet applications, the randomness quality is excellent.

## Syntax

> [!f(x)] RAND Syntax
>
> ```
> =RAND()
> ```

### Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| *(none)* | N/A | RAND takes no arguments. The empty parentheses are required. |

### Return Value

Returns a random decimal number greater than or equal to 0 and less than 1. The distribution is uniform—all values in the range are equally likely. Returns a new value each time the worksheet recalculates.

## Examples

> [!f(x)] RAND Examples

### Example 1: Basic Random Number
```
=RAND()
```
**Result:** A decimal like 0.583917264...

**Explanation:** Returns a random value between 0 and 1. Changes each time the sheet recalculates.

---

### Example 2: Random Number in Range 0 to 100
```
=RAND() * 100
```
**Result:** A decimal between 0 and 100

**Explanation:** Multiplying by 100 scales the 0-1 range to 0-100. Result is still a decimal.

---

### Example 3: Random Integer 1 to 10
```
=INT(RAND() * 10) + 1
```
**Result:** Integer from 1 to 10

**Explanation:** RAND()*10 gives 0 to 9.999..., INT truncates to 0-9, +1 shifts to 1-10. Note: RANDBETWEEN(1,10) is simpler.

---

### Example 4: Random Number in Custom Range
```
=Minimum + RAND() * (Maximum - Minimum)
```
**Result:** Random decimal between Minimum and Maximum

**Explanation:** For range 50 to 100: 50 + RAND()*50 gives values from 50 to 99.999...

---

### Example 5: Coin Flip Simulation
```
=IF(RAND() < 0.5, "Heads", "Tails")
```
**Result:** "Heads" or "Tails" with 50% probability each

**Explanation:** RAND()<0.5 is true about half the time. Creates a fair coin flip.

---

### Example 6: Dice Roll Simulation
```
=INT(RAND() * 6) + 1
```
**Result:** Integer from 1 to 6

**Explanation:** Simulates rolling a six-sided die. Each number has equal 1/6 probability.

---

### Example 7: Random Sampling (Yes/No with Custom Probability)
```
=IF(RAND() < 0.3, "Selected", "Not Selected")
```
**Result:** "Selected" 30% of the time

**Explanation:** Adjust the threshold (0.3) to change selection probability. Useful for random sampling.

---

### Example 8: Shuffle/Random Sort
```
=RAND()   (in helper column, then sort by it)
```
**Result:** Random ordering of data

**Explanation:** Add RAND() next to each row of data, sort by the RAND column, then delete the helper column. Data is now in random order.

---

### Example 9: Random Selection from List
```
=INDEX(List, INT(RAND() * COUNTA(List)) + 1)
```
**Result:** Randomly selected item from list

**Explanation:** Generates random index from 1 to list length, returns that item. Each item has equal probability.

---

### Example 10: Normal Distribution (Box-Muller Transform)
```
=SQRT(-2*LN(RAND()))*COS(2*PI()*RAND())
```
**Result:** Random number from standard normal distribution

**Explanation:** Transforms uniform random to normal distribution using Box-Muller method. Use NORM.INV(RAND(), Mean, StdDev) for a simpler approach.

## Common Errors

| Error | Cause | Solution |
|-------|-------|----------|
| `#NAME?` | Missing parentheses | Use RAND(), not RAND. Empty parentheses are required. |
| `Values keep changing` | Normal RAND behavior | RAND recalculates on every sheet change. Copy/paste values to preserve. |
| `Not truly random` | Pseudo-random generator | Acceptable for simulations; don't use for cryptography. |
| `Always same result` | Not actually RAND | Verify you're using RAND(), not a static value that looks random. |
| `Range incorrect` | Wrong transformation formula | For range [a,b]: =a + RAND()*(b-a). Common error is forgetting to add the minimum. |

## Use Cases

### [[Monte Carlo Simulation]]

**Scenario:** Simulate uncertain outcomes to estimate probabilities and distributions.

**Implementation:**
```
=Investment * (1 + NormalReturn + NORM.INV(RAND(),0,1)*Volatility)
=IF(RAND() < DefProb, "Default", "Paid")
=SUMPRODUCT(---(RAND() < SuccessProb))   (count successes in trials)
```

**Business Application:** Financial modeling, risk analysis, project planning, options pricing. Any scenario with uncertain inputs.

**Technical Details:** Run many iterations (often 1000+) with different random values each time. Analyze the distribution of outcomes. In Excel, use Data Tables or VBA for multiple iterations; in Sheets, consider scripts.

---

### [[Random Sampling and Selection]]

**Scenario:** Randomly select items from a population for surveys, testing, or auditing.

**Implementation:**
```
=IF(RAND() < SampleSize/PopulationSize, "Sample", "")
=SORT(Data, RANDARRAY(ROWS(Data)), 1)    (modern Excel)
=INDEX(Names, MATCH(MIN(RandColumn), RandColumn, 0))
```

**Business Application:** Survey sampling, quality control, audit selection, A/B testing assignment.

**Technical Details:** For simple random sampling without replacement, assign RAND() to each item, sort, take top n. For stratified sampling, apply within each stratum separately.

---

### [[Game and Educational Simulations]]

**Scenario:** Create random game elements or educational probability demonstrations.

**Implementation:**
```
=INT(RAND()*6)+1                          → Dice roll
=IF(RAND()<0.5,"Heads","Tails")           → Coin flip
=INDEX(Cards, INT(RAND()*52)+1)           → Random card
=CHOOSE(INT(RAND()*4)+1,"N","E","S","W")  → Random direction
```

**Business Application:** Educational tools, game prototypes, probability teaching, decision randomization.

**Technical Details:** Games requiring multiple dice can sum multiple RAND-based formulas. For card games, track which cards have been dealt to avoid duplicates.

## Platform Differences

### Microsoft Excel
- **Availability:** All versions since Excel 1.0
- **Algorithm:** Mersenne Twister (modern versions)
- **Recalculation:** Triggers on any cell change, F9, or manual recalc
- **RANDARRAY:** Excel 365 adds RANDARRAY for generating arrays of random numbers

### Google Sheets
- **Availability:** All versions
- **Algorithm:** Similar quality pseudo-random
- **Recalculation:** Same volatile behavior
- **RANDARRAY:** Not available; use array formulas with RAND() instead

### Both Platforms
- Same [0, 1) range
- Same volatile recalculation behavior
- Same pseudo-random quality
- Copy/paste values to preserve random values

## Tips and Best Practices

1. **Paste values to preserve:** RAND changes on every recalculation. If you need stable random values, immediately Copy > Paste Special > Values.

2. **Use RANDBETWEEN for integers:** RANDBETWEEN(1, 10) is cleaner than INT(RAND()*10)+1 for random integers.

3. **Transform for any range:** For decimals from a to b: =a + RAND()*(b-a). Remember the range is [a, b), not including b exactly.

4. **Combine multiple RANDs for distributions:** Box-Muller transform uses two RANDs to create normal distribution. Or use NORM.INV(RAND(), mean, stdev) directly.

5. **F9 forces recalculation:** Press F9 to generate new random values. Ctrl+Shift+F9 forces full recalculation.

6. **Consider RANDARRAY in modern Excel:** RANDARRAY(rows, cols, min, max, integer) generates entire arrays of random numbers efficiently.

---

```

```

```
=INDEX(List, INT(RAND()*ROWS(List))+1)
```

```

```

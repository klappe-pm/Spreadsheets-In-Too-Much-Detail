# LOGNORM.DIST

## Description

LOGNORM.DIST returns the lognormal distribution, a continuous probability distribution of a random variable whose logarithm is normally distributed. If X follows a lognormal distribution, then ln(X) follows a normal distribution. This makes the lognormal ideal for modeling variables that are always positive, cannot be negative, and result from multiplicative processes - like stock prices, incomes, city sizes, or biological measurements.

The distribution is characterized by two parameters: mean and standard_dev, which are the mean and standard deviation of the underlying normal distribution of ln(X), not of X itself. This is a crucial distinction. The lognormal distribution is always right-skewed, with a long tail extending toward positive infinity. Unlike the normal distribution, it is bounded below by zero, making it suitable for modeling inherently positive quantities.

Use LOGNORM.DIST for financial modeling (stock prices, option pricing via Black-Scholes), income and wealth distributions, product lifetimes in reliability engineering, environmental data (pollutant concentrations, rainfall), and any data generated by multiplicative random effects. It is essential when data shows positive values with right skew and you want to leverage the mathematical properties of the normal distribution through logarithmic transformation.

Both Excel and Google Sheets support LOGNORM.DIST with identical syntax, replacing the legacy LOGNORMDIST function. The function can return either the probability density function (PDF) or the cumulative distribution function (CDF). Note that the parameters refer to the log-transformed variable, not the original variable.

## Syntax

> [!info] LOGNORM.DIST Syntax
>
> ```excel
> =LOGNORM.DIST(x, mean, standard_dev, cumulative)
> ```

| Parameter | Description | Required |
|-----------|-------------|----------|
| `x` | The value at which to evaluate the distribution (must be > 0) | Yes |
| `mean` | The mean of ln(x), the natural log of x | Yes |
| `standard_dev` | The standard deviation of ln(x) (must be > 0) | Yes |
| `cumulative` | TRUE returns the cumulative distribution function; FALSE returns the probability density function | Yes |

## Examples

```excel
=LOGNORM.DIST(10, 2, 0.5, TRUE)
```
**Result:** `0.3413` (approximately)
**Explanation:** Returns the probability that a lognormal variable (with ln(X) having mean=2, sd=0.5) is less than or equal to 10. About 34% of values fall at or below 10.

```excel
=LOGNORM.DIST(10, 2, 0.5, FALSE)
```
**Result:** `0.0487` (approximately)
**Explanation:** Returns the probability density at x=10. This is the height of the PDF curve at that point, useful for plotting.

```excel
=LOGNORM.DIST(EXP(2), 2, 0.5, TRUE)
```
**Result:** `0.5`
**Explanation:** When x = e^mean, the cumulative probability is always 0.5 (the median). Here, e^2 = 7.389, and half of values fall below this.

```excel
=LOGNORM.DIST(100, 4, 1, TRUE)
```
**Result:** `0.7899` (approximately)
**Explanation:** About 79% of values from this lognormal distribution are 100 or less. The mean of ln(X) is 4, so the median is e^4 = 54.6.

```excel
=1 - LOGNORM.DIST(50, 3, 0.8, TRUE)
```
**Result:** `0.1676` (approximately)
**Explanation:** Probability of exceeding 50 when ln(X) has mean=3 and sd=0.8. About 16.8% of values exceed 50.

```excel
=LOGNORM.DIST(200, 4.5, 0.6, TRUE) - LOGNORM.DIST(100, 4.5, 0.6, TRUE)
```
**Result:** `0.2827` (approximately)
**Explanation:** Probability of falling between 100 and 200. About 28% of values fall in this range.

```excel
=LOGNORM.DIST(1, 0, 1, TRUE)
```
**Result:** `0.5`
**Explanation:** When mean=0, the median equals e^0=1. This is the standard lognormal distribution, analogous to the standard normal.

```excel
=LOGNORM.DIST(A2, AVERAGE(LN(A:A)), STDEV.S(LN(A:A)), TRUE)
```
**Result:** (Varies based on data)
**Explanation:** Fits a lognormal to actual data by computing the mean and standard deviation of the log-transformed values. Returns the percentile of A2 in this distribution.

```excel
=LOGNORM.DIST(B2, LN(Median), C2, TRUE)
```
**Result:** (Varies based on values)
**Explanation:** Uses the relationship that mean of ln(X) equals ln(median of X). If you know the median directly, you can derive the mean parameter.

```excel
=NORM.DIST(LN(10), 2, 0.5, TRUE)
```
**Result:** `0.3413` (same as first example)
**Explanation:** LOGNORM.DIST(x, mean, sd, TRUE) equals NORM.DIST(LN(x), mean, sd, TRUE). This demonstrates the relationship between the two distributions.

## Common Errors

| Error | Cause | Solution |
|-------|-------|----------|
| `#NUM!` | x is zero or negative | Lognormal is only defined for positive x values |
| `#NUM!` | standard_dev is zero or negative | Standard deviation must be positive |
| `#VALUE!` | Non-numeric arguments provided | Ensure all arguments are numbers or cell references |
| `Conceptual error` | Confusing parameters | Mean and standard_dev refer to ln(X), not X itself |
| `Unexpected results` | Using arithmetic mean/SD | Calculate mean and SD of log-transformed data |

## Use Cases

### Stock Price Modeling

**Scenario:** A financial analyst models stock prices assuming they follow a lognormal distribution. Current stock price is $100, and historical data shows ln(price changes) have mean 0.1 (10% expected growth) and standard deviation 0.3 per year. What is the probability the stock exceeds $150 in one year?

**Implementation:** Calculate probability of price > $150:
```excel
=1 - LOGNORM.DIST(150/100, 0.1, 0.3, TRUE)
```
The ratio 150/100 = 1.5 represents a 50% gain. This returns approximately 0.188, meaning about 19% probability of exceeding $150.

**Business Impact:** The lognormal assumption underlies the Black-Scholes option pricing model. Understanding the probability distribution of future prices enables informed trading decisions and risk management.

### Income Distribution Analysis

**Scenario:** A city planner analyzes household incomes that follow a lognormal distribution. Historical data shows ln(income) has mean 10.8 (median income around $49,000) and standard deviation 0.9. What percentage of households earn between $30,000 and $100,000?

**Implementation:** Calculate probability of income in range:
```excel
=LOGNORM.DIST(100000, 10.8, 0.9, TRUE) - LOGNORM.DIST(30000, 10.8, 0.9, TRUE)
```
This returns approximately 0.592, meaning about 59% of households fall in this income range.

**Business Impact:** Income distribution modeling supports tax policy analysis, housing demand forecasting, and social program planning. The lognormal captures the characteristic right skew of income data.

### Product Lifetime Analysis

**Scenario:** A reliability engineer models the lifetime of LED bulbs. Testing shows lifetimes follow a lognormal distribution with ln(hours) having mean=9 (median ~8,100 hours) and standard deviation=0.5. What percentage of bulbs fail before 5,000 hours?

**Implementation:** Calculate early failure probability:
```excel
=LOGNORM.DIST(5000, 9, 0.5, TRUE)
```
This returns approximately 0.131, meaning about 13% of bulbs fail before 5,000 hours.

**Business Impact:** Lifetime distribution analysis supports warranty decisions, replacement inventory planning, and quality improvement priorities. The lognormal often fits failure time data better than exponential or normal distributions.

## Tips and Best Practices

1. **Remember parameters describe ln(X):** The mean and standard_dev are NOT the mean and SD of X. They are the mean and SD of ln(X). This is the most common source of confusion.

2. **Calculate parameters from data:** If you have raw data, compute parameters as AVERAGE(LN(data)) and STDEV.S(LN(data)).

3. **Use the median relationship:** Median of X = exp(mean), where mean is the first parameter. This provides intuitive interpretation.

4. **Relate to normal distribution:** LOGNORM.DIST(x, m, s, TRUE) = NORM.DIST(LN(x), m, s, TRUE). Use this to verify calculations.

5. **The mean of X is NOT exp(mean parameter):** The actual mean of X = exp(mean + standard_dev^2/2). The median exp(mean) is usually more interpretable.

6. **Use for multiplicative processes:** When values result from multiplying many factors (like compound growth), the lognormal often fits well because logs of products are sums.

### Commonly Used Together

**[[LOGNORM.INV]]** - Find percentiles and thresholds

*Calculate the 95th percentile:*
```excel
=LOGNORM.INV(0.95, 4, 0.5)
```
Returns the value below which 95% of the distribution falls.

**[[LN]]** - Transform data for parameter estimation

*Fit lognormal to data:*
```excel
Mean: =AVERAGE(LN(data))
SD: =STDEV.S(LN(data))
```
These estimated parameters can then be used in LOGNORM.DIST.

**[[NORM.DIST]]** - Verify using normal relationship

*Cross-check calculation:*
```excel
=NORM.DIST(LN(x), mean, standard_dev, TRUE)
```
This equals LOGNORM.DIST(x, mean, standard_dev, TRUE).

# XLOOKUP

## XLOOKUP Description

Searches a range or array and returns the corresponding value from another range or array. Modern replacement for VLOOKUP with more flexibility.

> [!f(x)] XLOOKUP Syntax
>
> ```spreadsheets
> XLOOKUP()
> ```

> [!f(x)] XLOOKUP Example
>
> ```spreadsheets
> XLOOKUP() → result
> ```

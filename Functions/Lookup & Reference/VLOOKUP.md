# VLOOKUP

## VLOOKUP Description

Searches vertically for a value in the first column and returns a value from a specified column

> [!f(x)] VLOOKUP Syntax
>
> ```spreadsheets
> VLOOKUP()
> ```

> [!f(x)] VLOOKUP Example
>
> ```spreadsheets
> VLOOKUP() → result
> ```

## How to View
These diagrams use Mermaid.js syntax. To view them:
1. Use any Markdown viewer that supports Mermaid (like GitHub, GitLab, Obsidian with Mermaid plugin)
2. Copy the diagram code into [Mermaid Live Editor](https://mermaid.live/)
3. Use VS Code with Mermaid preview extension

## Diagram Legend
- **Solid arrows (→)**: "Commonly used with" relationships
- **Dashed arrows (-.->)**: "Similar functions" relationships
- **Node colors**: Represent function categories
- **External nodes**: Functions from other categories (shown with dashed borders)
